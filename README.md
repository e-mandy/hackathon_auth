# Authentification Management React + Typescript

## OBJECTIFS DU PROJET

L'objectif de ce projet était de gérer le flow complet d'une authentification en React TS. 
A travers ce projet, le développeur a 
  - mieux compris le rôle de chacun des éléments qui entrent en jeu dans le processus d'authentification; 
  - l'ordre d'enchainement des actions 
  - ainsi que les priorités de réalisation lors de la phase de développement.

## Technologies utilisées

  1. React Hook Form
  2. Zustand
  3. Zod
  4. React Query
  5. React Router
  6. React Testing Library & Vitest (pour les tests)


## COMMENT ACCÉDER AU PROJET

  1. Cloner le repository

    git clone git@github.com:e-mandy/hackathon_auth.git


  2. Installer les différents packages

    npm install
  
  3. Lancer le serveur

    npm run dev
  

  4. Visiter le lien

    http://localhost:5173
